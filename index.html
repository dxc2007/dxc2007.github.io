Machine Learning ~ Databases ~ Web Development

Hi, I am Xiaochen.

I'm a double major in computer science and chemistry at Duke University.

I started programming after graduating from high school. My first interest was in web development, in which I learnt basic HTML, CSS and JavaScript before moving on to the full-stack using ReactJS, NodeJS and MongoDB.

In my first year of college, I created an online database for a materials science lab. Eventually, I saw the true value of the database was in using data analysis to discover trends and perhaps uncover new material properties.

That marked my transition to machine learning. Using neural networks, I'm attempting to develop new methods of doing chemistry research.

In the future, I aim to continue applying machine learning and other data analysis methods to gain a better understanding of science.

Apart from programming and chemistry , I'm also interested in math, physics and statistics.

When I'm free, I like to run. Especially long distances.

Machine Learning for quantum waves [WIP]

Technologies: Keras with TensorFlow, Anaconda Python

HybriD3 Materials Database

Database curates and desseminates experimental and calculated results of perovskites -- a promising group of materials for next-gen solar panels. This database was designed from scratch and hosted with an Apache server on a RHEL machine. Future plans include using machine learning techniques to identify trends within the collated data and chart out new research directions for both experimental and theory groups.

Technologies: Python Django, MariaDB, Bootstrap, RHEL + Apache
